# À la découverte des singes

## URL du site
- URL en ligne : http://elshennawy.projetsmmichamps.fr/sae105/

## Installation locale avec XAMPP
1. Copier le dossier du site dans le dossier `htdocs` de XAMPP.
2. Démarrer le module **Apache** dans le panneau de contrôle XAMPP.
3. Dans le navigateur, aller sur : `http://localhost/[nom_du_dossier]/`.

## Fonctionnalités principales
- Affichage d’une galerie de singes à partir d’un fichier JSON.
- Popup permettant d’agrandir les images.
- Formulaire pour créer une fiche de singe avec aperçu en temps réel.
- Envoi des informations du formulaire vers une API avec un message contenant le titre, la description, le choix et l’URL de l’image.
